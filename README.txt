日常打卡 & 日记 使用说明
================================

## 打开方式
双击 index.html 即可在浏览器中使用。

## 功能概览

### 打卡
- 点击 + 按钮创建打卡项目（名称、emoji图标、最大数值）
- 点击 emoji 图标循环切换数值（0 → 1 → 2 → ... → 最大 → 0）
- 备注在输入框失焦时自动保存
- 日期前后箭头切换查看不同天

### 日记
- 在文本框写日记，失焦自动保存
- 点击上传区或拖拽图片到上传区即可附图
- 悬停图片显示删除按钮

### 统计
- 切换"本周"/"本月"查看打卡图表
- 显示打卡率、最长连续天数等汇总数据
- 点击"生成周总结"按钮，调用 AI 生成一周总结
  - 需要先设置 Anthropic API Key（首次会弹出提示输入）
  - Key 仅保存在本地浏览器 localStorage 中

### 历史
- 按日期范围和项目筛选历史记录
- 点击行跳转到对应日期
- 导出为 CSV 文件

## 数据存储
所有数据存储在浏览器本地（localStorage + IndexedDB），不会上传到任何服务器。
清除浏览器数据会丢失记录，建议定期导出 CSV 备份。
